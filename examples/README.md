# @cloudy/examples

Here you can try some example stacks.

## Deploying Examples

Just run `cd examples` and:

```sh
# Deploy example 1:
pnpx cdk deploy --app "esm-node src/1-hello-world/index.ts"

# Deploy example 3:
pnpx cdk deploy --app "esm-node src/3-publish-to-topic/index.ts"

# And so on...
```
